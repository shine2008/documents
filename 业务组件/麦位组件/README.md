# 概述

## 产品介绍

麦位组件产品介绍

## 功能特性

| 功能        | 描述                                                         |
| :---------- | ------------------------------------------------------------ |
| 申请麦位    | 观众申请麦位，主播同意或拒接                     |
| 抱麦 | 主播将观众抱上麦，观众可以同意或拒绝 |
| 麦位管理 | 观众可以开关自己的摄像头或者麦克风，主播可以开关所有人的                       |



## 产品架构

<img src="./image/麦位组件整体方案.png" width="50%" />

麦位组件基于云信NIM SDK的基础能力，封装了业务侧的麦位相关请求，并解析服务端的相关消息协议，通过delegate 回调给业务层，完成麦位管理
